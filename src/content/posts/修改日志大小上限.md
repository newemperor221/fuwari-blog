# 修改日志大小上限

```bash
sudo vim /etc/systemd/journald.conf
```

修改以下参数

```bash
SystemMaxUse=200M
SystemKeepFree=500M
MaxRetentionSec=7day
```

重启服务

```bash
sudo systemctl restart systemd-journald
```
